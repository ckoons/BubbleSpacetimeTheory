## Why these are preserved

- **Reference value**: a future reader (or future Casey) may want to see the v34 abstract framing or the pre-split monolithic structure
- **Citation continuity**: any external citation to "Working Paper v34" or "Working Paper v36 monolithic" has a stable target here
- **Disk space is cheap**: total here is ~3.3 MB; preserving history costs almost nothing
- **Casey directive 2026-05-19**: "delete or archive if mentioned somewhere logical" — this README is that mention

## Not preserved

The colon in the original filenames (`Bubble_Spacetime_Theory:Five_Integers,Zero_Free_Parameters.md`) was problematic for cross-platform git use (Windows rejects colons). The renamed versions in this archive use underscores. The original colon-form filenames are documented in git history.

## When to delete

If disk space ever matters or the archive grows beyond ~50 MB, the v34-era files can be deleted; the v36 monolithic and the v35 PDF are more recent and worth keeping until at least v40 (whenever that lands).
