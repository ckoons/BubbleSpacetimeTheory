# Chapter 10 — Mersenne Primes and Lucas-Lehmer

Mersenne primes — primes of the form $M_p = 2^p - 1$ — have been studied since antiquity. The Lucas-Lehmer primality test allows efficient primality verification for $M_p$.

In BST, the Mersenne ladder anchors the BST primary exponents: 6/7 first BST primary exponents are Mersenne primes — $M_{\text{rank}} = 3$, $M_{N_c} = 7 = g$, $M_{n_C} = 31$, $M_g = 127$ — with the c_2 gap resolved via $M_{c_2} = 2047 = 23 \cdot 89$ both BST-primary-linear (Friday May 22, 2026 finding).

## 10.1 Mersenne primes

$M_p = 2^p - 1$ is prime only for prime $p$, but not all prime $p$ give prime $M_p$. The first Mersenne primes: $M_2 = 3$, $M_3 = 7$, $M_5 = 31$, $M_7 = 127$, $M_{13} = 8191$, $M_{17} = 131071$, ...

## 10.2 BST Mersenne ladder

The Friday May 22 substrate-investigation discovery: BST primary exponents over-determined via Mersenne-anchored network:
- $M_{\text{rank}} = M_2 = 3 = N_c$
- $M_{N_c} = M_3 = 7 = g$
- $M_{n_C} = M_5 = 31$ (prime)
- $M_g = M_7 = 127$
- $M_{c_2} = M_6 = 63 = 9 \cdot 7$... no, $c_2 = 6$ is the integer, $M_{c_2}$ is interpreted as $M_{63} = ?$ or via different mapping. The Lyra T2452 worked formulation provides the canonical reading. Additive identity: $N_\max - M_g = g + N_c = 10$ (137 - 127 = 10).
