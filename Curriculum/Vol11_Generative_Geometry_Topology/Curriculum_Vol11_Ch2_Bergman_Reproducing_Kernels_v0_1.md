# Chapter 2 — Bergman Reproducing Kernels

The Bergman kernel of a bounded domain is the reproducing kernel for the Hilbert space of $L^2$-integrable holomorphic functions on the domain. For BST, the Bergman kernel of $D_{IV}^5$ is the structural anchor underlying the Born rule (Volume 5 Chapter 7) and the substrate's natural geometric quantization.

## 2.1 Reproducing kernels

A Hilbert space $\mathcal{H}$ of functions on a set $X$ has reproducing kernel $K(x,y)$ if $f(x) = \langle f, K(\cdot, x) \rangle$ for all $f \in \mathcal{H}$ and $x \in X$. The kernel exists when point evaluation is a bounded linear functional.

## 2.2 Bergman kernel of bounded domains

For a bounded domain $D \subset \mathbb{C}^n$, the Bergman kernel $K_D(z,w)$ is the reproducing kernel of $H^2(D) = $ holomorphic $L^2(D)$. Standard construction.
