### 5.1 K-audit anatomy

Each K-audit file includes:
- **K-number**: K21, K36, K57 (sequential)
- **Scope**: what's being audited (proof, paper, claim, framework)
- **F1-F4 ratings**: form / formal / framework / falsifiability
- **B1-B4 ratings** (Phase 2): backbone / boundary / breadth / bridges
- **Verdict**: PASS / CONDITIONAL PASS / FAIL / RATIFIED
- **Confidence**: ~%
- **Gaps identified** (if any)
- **Severity**: CRITICAL / MODERATE / MINOR

### 5.3 Tier ladder

- **CANDIDATE**: claim filed
- **PRE-STAGE**: empirical + form check
- **STRUCTURALLY VERIFIED**: independent corroboration
- **RIGOROUSLY CLOSED**: empirical + mechanism + form
- **RATIFIED**: K-audit chain consensus
- **PUBLISHED**: external dispatch

Each step gated by Cal referee + Keeper PASS.
