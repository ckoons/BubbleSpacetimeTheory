# Chapter 7 — Atomic Orbital Sequence

The atomic-orbital structure of multielectron atoms follows the famous sequence $1s, 2s, 2p, 3s, 3p, 3d, 4s, \ldots$ — with orbital occupancies of $(2l+1) = 1, 3, 5, 7$ electron-pairs per shell type. The pattern is empirical in standard atomic physics; it emerges from the hydrogenic shell structure modified by electron-electron interactions.

BST identifies the orbital sequence with BST primary integers exactly:

$$(2l+1) \;=\; 1, \; N_c, \; n_C, \; g \;=\; 1, \; 3, \; 5, \; 7.$$

## 7.1 The structural identification

The four atomic orbital types ($s, p, d, f$) correspond to angular-momentum quantum numbers $l = 0, 1, 2, 3$. The orbital occupancies $(2l+1)$ give $1, 3, 5, 7$ — exactly the sequence $1, N_c, n_C, g$ of BST primaries (with the trivial $1$ corresponding to the $s$-orbital scalar K-type).

This identification is structurally significant. The atomic shell structure is not separate physics — it is the substrate's K-type angular-momentum spectrum applied at the atomic energy scale.

## 7.2 Consequence: the periodic table

The periodic table's structure — eight-column blocks for $s+p$ (8 electrons), eighteen-column blocks for $s+p+d$ (18 electrons), thirty-two-column blocks for $s+p+d+f$ (32 electrons) — emerges directly from the substrate's $1, 3, 5, 7$ K-type degeneracies. The periodic table is the substrate's orbital-spectrum at the atomic scale.
