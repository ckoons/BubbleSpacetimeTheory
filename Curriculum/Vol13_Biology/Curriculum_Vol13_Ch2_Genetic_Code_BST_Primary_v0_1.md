## Section 2.1 — DNA Structure (Watson-Crick Double Helix)

Two anti-parallel strands; sugar-phosphate backbone; 4 nucleotides projected inward forming base pairs.

Base pair geometry:
- A-T: 2 hydrogen bonds (weaker)
- G-C: 3 hydrogen bonds (stronger)

GC-content variation across genomes affects melting temperature (higher GC → higher T_m via more H-bonds).

BST framing: 4 nucleotides = 2^rank; H-bond differential (2 vs 3) reads BST primary integer cluster (rank, N_c).

## Section 2.2 — Codon Table and 64-Codon Structure

64 = 4³ codons (3-nucleotide codon length).
61 sense codons (encode amino acids).
3 stop codons (UAA, UAG, UGA; "amber, ochre, opal").

Start codon AUG encodes methionine; canonical translation initiation.

Codon frequency varies across organisms (codon usage bias); affects translation efficiency + protein expression in heterologous systems.

BST framing: 64 = 2^C_2 where C_2 = 6; codon length 3 = N_c BST primary integer.

## Section 2.3 — 20 Amino Acids and Degeneracy

20 standard amino acids; 61 sense codons → degeneracy.

Amino acid count distribution:
- 1 codon: Met, Trp
- 2 codons: Phe, Tyr, His, Gln, Asn, Lys, Asp, Glu, Cys
- 3 codons: Ile
- 4 codons: Val, Pro, Thr, Ala, Gly
- 6 codons: Leu, Ser, Arg

Three-letter pattern in degeneracy: most amino acids have codons differing only at position 3 (synonymous-codon cluster).

BST framing: 20 ≈ N_max/g = 137/7 = 19.6; 20 - 19.6 = 0.4 = wobble-codon correction factor. Six-codon amino acids (Leu, Ser, Arg) = C_2 codons per amino acid.

## Section 2.4 — Wobble Hypothesis (Crick 1966)

Crick wobble hypothesis: position 3 of codon allows non-Watson-Crick pairing with anticodon position 1; modified bases at tRNA anticodon (inosine, modified U) enable one tRNA to read multiple codons.

Reduces tRNA species count: instead of 61 tRNAs (one per sense codon), ~31 tRNA species suffice.

BST framing: wobble flexibility at position 3 = substrate K-type tolerance for one-position substitution; tRNA species count ~31 ≈ 2^N_c+ correction.

## Section 2.5 — Universal vs Variant Genetic Codes

Standard code: nearly universal across all organisms.

Variants:
- Mitochondrial codes: UGA = Trp (not stop); AGA + AGG = stop (not Arg) in vertebrate mito
- Ciliate codes: UAA + UAG = Gln (not stop) in some ciliates
- Pyrrolysine + selenocysteine: UAG → Pyl, UGA → Sec in special contexts

Variations are minor; near-universality suggests early common ancestor with frozen code (Crick "frozen accident" hypothesis).

BST framing: code universality reflects substrate-forcing of integer combinatorial structure; variants are local re-readings.
