### Concrete checklist for any external document

Before any document moves to external/venue/outreach status:
1. **Criterion count**: explicitly state "current ratified count" (10 FORMAL + 1 ASPIRATIONAL + N candidates) — never "11-15"
2. **Null model**: only use joint arithmetic over individually-RIGOROUSLY-CLOSED criteria; never include SEED/STRUCTURALLY VERIFIED as ratified
3. **Methodology tier**: each invoked theorem must show its current tier (RIGOROUSLY CLOSED / RATIFIED / STRUCTURALLY VERIFIED / SEED / CANDIDATE)
4. **Honest-scope refinements**: T2462 / T2459 honest-scope must appear in narrative, not buried in registry
5. **Header claims**: paper section headers must match the tier-level framing within (no "Bergman = Feynman propagator" header backed by "structural role of" body text)
