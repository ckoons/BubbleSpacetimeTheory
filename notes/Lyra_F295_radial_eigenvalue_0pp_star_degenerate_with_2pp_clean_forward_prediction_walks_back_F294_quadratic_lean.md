## The clean computation

The first excited scalar glueball 0⁺⁺* is the first **radial** excitation of the ground 0⁺⁺ — the K-type (m_1, m_2) = (1,1), one unit of the invariant r². The key rep-theory fact: **within a single discrete-series irrep the quadratic Casimir is constant.** So the (1,1) radial state and the (0,0) ground share the same Casimir; the only quantum number that moves is the SO(2) conformal energy:

  **E(0⁺⁺*) = λ_0 + (m_1 + m_2) = n_C + 2 = 7.**

The 2⁺⁺ ground state is the K-type (2,0), also at E = n_C + 2 = 7. So the **same linear operator, with no new inputs**, predicts:

  **0⁺⁺* degenerate with 2⁺⁺: m(0⁺⁺*) ≈ m(2⁺⁺) ≈ 2420 MeV.**
