## The completion — the SO(2)=J carries TWO orientations, and C is the second
The SO(2) = J (D_IV⁵ Hermitian) provides two distinct orientations, and the discrete symmetries are the flips of each:
- **P (parity) = the SPATIAL orientation flip** — self-dual ↔ anti-self-dual, i.e. SU(2)_L ↔ SU(2)_R (Kähler canonical orientation fixes which is which, K811/K813). **Maximally violated:** weak = SU(2)_L only. *L → R.*
- **C (charge conjugation) = the COMPLEX-structure flip J → −J** — particle ↔ antiparticle (complex conjugation of the fields). **Maximally violated BECAUSE the weak SU(2)_L is J-CHARGED** (J-charges {0,±2}, Lyra's computation): a J-charged gauge coupling is not invariant under J → −J. *L → L̄.* (Contrast: SU(2)_R is J-neutral {0,0,0} → *would* be C-symmetric — but it's ungauged. So the one gauged factor is exactly the C-violating one.)
- **CP = both flips** — *L → R̄.* **Nearly conserved:** the weak force couples to both the left-particle AND the right-antiparticle (the CP pair), so CP maps weak-states to weak-states. The ONLY residual that violates CP is the **continuous phase δ → FREE** (K810/K811). Discrete P and C violations combine and cancel in CP; only the continuous phase survives.
- **T (time reversal) ⟺ CP** — by CPT (below), T-violation = CP-violation = the free phase (microscopic T; NOT the thermodynamic arrow — K810 flag).
- **CPT conserved** — inherited from **Lorentz invariance** (SO(5,2) → SO(3,1), the physical Lorentz), + locality + unitarity. The full flip (all orientations + T) closes to the identity.
