## 1. The placement: shared boundary generations, bulk color fiber

The key structural fact, stated plainly:

> **The three-generation structure is the BOUNDARY stratification of D_IV⁵ — and it is flavor-blind, hence shared by quarks and leptons. Color is the BULK fiber.**

- **Generations = Korányi-Wolf strata = Wallach points.** F88/F92/F93: the closure of D_IV⁵ has rank+1 = 3 boundary strata, carrying the three generation parameters ν ∈ {n_C/rank, N_c/rank, 0} = {5/2, 3/2, 0} (the two ρ-components + 0). This stratification is a property of the **domain geometry**, with no reference to color or charge. So it is the SAME three ν's for every fermion tower. This is exactly F86's "quark + lepton 3-generation matching falls out from the same D_IV⁵."
- **Color = the bulk characteristic multiplicity a = N_c.** F92 computed a = n_C − 2 = 3 = N_c — the domain's characteristic multiplicity (which governs the entire bulk degeneration structure) IS the color number. Color therefore lives in the **bulk multiplicity**, not on the boundary strata. The SU(3) color fiber rides on top of the (flavor-blind) boundary placement.

So the picture is a **product**: (boundary generation ν) × (bulk color fiber). A **lepton** is a **color-singlet** — trivial fiber — and lands cleanly on its boundary stratum (the clean Wallach-point norm the lepton-mass program uses). A **quark** carries the **fundamental** color fiber.

## 2. Why this IS confinement (and matches F77)

The boundary strata are **color-blind** — they are degenerations of the domain geometry, carrying no color index. A state can terminate (become a free asymptotic particle, "land on the boundary") only if its color fiber is **trivial there**. So:

- A **single quark** (fundamental fiber, nontrivial) **cannot terminate on the color-blind boundary.** It has no clean boundary endpoint → no clean boundary norm → no scale-free mass. This is **confinement, read as geometry**: only **color-singlet combinations** (q̄q mesons, qqq baryons — where the fundamental fibers contract to a singlet) can reach the boundary.
- This is precisely **F77** (Monday's result): the confined single quark is a scale-dependent K-type superposition with sliding occupied volume — *no single value for the substrate to measure*. F94 says the same thing in the boundary language: no clean boundary norm because the color fiber forbids termination. Two routes, one fact.

## 4. What this gives the fork — the clean anchor

Naively "use the quark sector" stumbles on Section 2: absolute quark masses are bulk-contaminated (scale-dependent), so they are **not** clean test data for "algebraic vs log." But two things **are** clean, and they make the anchor well-posed:

**(a) Within-type mass ratios are RG-invariant and sit at the same BF point.** The mass anomalous dimension in QCD is **flavor-blind** (depends only on color charge), so for same-charge quarks the ratio m_c/m_u (up-type 2nd/1st) and m_s/m_d (down-type 2nd/1st) are **RG-invariant at one loop** — the running cancels in the ratio. These ratios are clean. And the lightest member of each tower (u, d) sits at the **same BF point ν = 5/2 = d/2** as the electron (shared boundary generations, Section 1), so each ratio's denominator inherits whatever the BF point does. Therefore:

> The three "second-over-first" ratios **{m_μ/m_e, m_c/m_u, m_s/m_d}** all test the **same** structural prediction — does the BF point put a **log** in the coefficient (K310) or an **algebraic π-form** (2π⁴+12)? The lepton sector gives **one** such ratio (undecidable, 3 masses → 2 ratios). The quark sector adds **two more**, each RG-clean. **3× the constraint** on one functional form.

This does not *trivially* decide the fork — the three ratios are different numbers (≈207, ≈577, ≈20), so the test is "does **one** functional form f(BF point; tower data) — log-carrying or algebraic — reproduce all three at once," which needs the mass-map (still open). But it **triples** the constraints an algebraic-vs-log form must satisfy simultaneously, which is exactly the overdetermination the lepton sector lacks. A log-in-the-coefficient form and a π-algebraic form will generically diverge across three towers even where they agree on one.

**(b) CKM is the mixing-axis cross-check (π-free).** Per the trichotomy (Toy 4045 / Cal #237): mass = measuring (π-carrying), mixing = counting (π-free). The CKM is the up↔down boundary-eigenvector overlap. The BF feature is a **mass-axis** (measuring) signature, so its imprint on the **mixing** texture (the Wolfenstein λ hierarchy) is an independent, π-free cross-check — and the CKM is measured very precisely. PMNS (large) vs CKM (small) is the same axis on the lepton side.
