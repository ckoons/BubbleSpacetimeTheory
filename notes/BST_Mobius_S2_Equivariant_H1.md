# Gap #2 Session 2: Equivariant H¹ Computation

## Reframing from Session 1

Session 1 established that M(D_IV⁵) ≅ open 5-ball — contractible. So the ABSOLUTE cohomology of M is trivial: H^k(M, Z) = 0 for k > 0. **There is no Z/2 in H¹ of M itself.**

The Z/2 orientation class must come from somewhere else. Session 2 identifies the right framework: **equivariant cohomology of the Z/2 action τ on D_IV⁵, restricted to M.**

This is sharper than the Mayer-Vietoris setup scoped for Session 2 in v0.2. The computation is shorter; the conceptual landing is clearer.

## Equivariant cohomology setup

The involution τ: D_IV⁵ → D_IV⁵, τ(z) = z̄, generates a Z/2 group action on D_IV⁵. The fixed-point set of this action is M(D_IV⁵) (Session 1 result). On M itself, the action is **trivial** (every point of M is fixed).

For a group G acting on a space X, the equivariant cohomology is:
```
H*_G(X, Z) := H*(EG ×_G X, Z)
```
where EG is a contractible space with free G-action and EG ×_G X = (EG × X) / G.

For G = Z/2: EG = S^∞ (with antipodal action), so BG = ℝP^∞ (real projective infinite space).

## Computation of H*_{Z/2}(M, Z)

Since the Z/2 action on M is trivial:
```
EG ×_G M = (EG × M) / G = (EG / G) × M = ℝP^∞ × M
```

Since M is contractible:
```
ℝP^∞ × M ≃ ℝP^∞
```

Therefore:
```
H*_{Z/2}(M, Z) = H*(ℝP^∞, Z)
```

The cohomology of infinite real projective space is a classical computation:
```
H*(ℝP^∞, Z) = Z[w] / (2w)    where deg(w) = 1
```

i.e.:
- H⁰(ℝP^∞, Z) = Z  (generated by 1)
- H¹(ℝP^∞, Z) = Z/2  (generated by w)
- H²(ℝP^∞, Z) = 0  (since 2w = 0 in Z coefficients)
- H³(ℝP^∞, Z) = Z/2  (generated by w³)
- H^k(ℝP^∞, Z) = Z/2 for k odd, 0 for k > 0 even

**Conclusion**:
```
H¹_{Z/2}(M(D_IV⁵), Z) = Z/2
```

generated by the class w ∈ H¹(BZ/2, Z). This is the **orientation Z/2 class** — the "Möbius orientation" of the v0.1 sketch.

## What this means physically

The Z/2 orientation class is NOT a feature of M itself (which is contractible and has trivial cohomology). It is a feature of the **Z/2 symmetry** (the complex-conjugation involution τ) that acts on D_IV⁵ and fixes M pointwise.

In physics language:
- The Z/2 group is the **CP symmetry** of the D_IV⁵ structure (T1947)
- Its H¹ class w is the **CP-charge** of operators in the cohomology
- An operator inheriting w gets a sign change under CP → this IS the "Möbius -1"

So the **Möbius -1 in lepton mass formulas (T2003/T2091) is the equivariant CP-charge of lepton operators inhabiting M(D_IV⁵).**

## Result statement (rigorous)

**Theorem (Lyra Session 2, 2026-05-17)**:

Let D_IV⁵ = SO_0(5,2)/[SO(5)×SO(2)] be the Hermitian symmetric domain of Cartan type IV in 5 complex dimensions. Let τ: D_IV⁵ → D_IV⁵ be the anti-holomorphic involution τ(z) = z̄ in Hua coordinates. Let M = M(D_IV⁵) = Fix(τ) ≅ open 5-ball in ℝ⁵ (per Session 1). Let G = ⟨τ⟩ = Z/2.

Then:
```
H¹_G(M, Z) = Z/2
```

generated by the pullback of w ∈ H¹(BG, Z), and this Z/2 class is the **Möbius orientation class** carried by all G-equivariant operators on M.

**Proof**: G acts trivially on M (every point is fixed by τ). Therefore EG ×_G M = (EG/G) × M = BG × M = ℝP^∞ × M. Since M is contractible (Session 1), this is homotopy-equivalent to ℝP^∞. Hence H*_G(M, Z) = H*(ℝP^∞, Z) = Z[w]/(2w) with deg(w)=1, so H¹ = Z/2. □

This is a rigorous statement, deriving from established machinery (equivariant cohomology, classical computation of H*(ℝP^∞, Z), Session 1's identification of M as contractible).
