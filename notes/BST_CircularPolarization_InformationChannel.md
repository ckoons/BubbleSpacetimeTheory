## 3. Information-Theoretic Framework

If circular polarization carries geometric information, then photons

### 3.1 Protocol vs Message

Following standard communication theory, any structured signal
has two components:

- **Protocol** (structural, universal): features that are the same
  regardless of source — channel identification, framing, error
  structure. Analogous to packet headers.
- **Message** (content, source-dependent): features that vary with
  the emitting geometry — curvature magnitude, local topology.
  Analogous to packet payload.

Separation criterion:
- Protocol features should be UNIVERSAL across all sources
- Message features should CORRELATE with independently measured
  source properties (mass, compactness, curvature)

### 3.2 Error Correction Prediction

Shannon's noisy channel theorem: any real information channel
operating above its noise floor requires error correction to
maintain fidelity. If circular polarization is a real geometric
information channel, then:

- The signal should contain redundancy structure
- Redundancy level should correlate with channel noise
  (higher near noisy environments, lower in quiet regions)
- Statistical properties should deviate from random

### 3.3 Loosely Coupled Transport

The natural transport model is loosely coupled message passing:
- Each photon is self-contained (no handshake required)
- Messages may be lost without system failure (redundancy)
- Latency is path-dependent (speed of light, gravitational delay)
- Global coherence emerges from local propagation

This is consistent with the observed locality of physics —
information propagates at c, interactions are local, yet
large-scale structure is coherent.
