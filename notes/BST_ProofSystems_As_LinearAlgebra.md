# Proof systems as linear algebra

Casey's question: write what we have as linear algebra, write EF as linear algebra,
and see what intermediary steps connect them. Doing it locates the wall exactly.

## 1. What we have, as linear algebra (bounded-degree certificates)

Encode the CNF `phi'` (the residual expander) as polynomial axioms: each clause
`C` becomes a polynomial `p_C` (e.g. `prod_{l in C}(1 - l) = 0`), plus Boolean axioms
`x_i^2 - x_i = 0`. `phi'` is UNSAT iff `1` lies in the ideal generated by the `p_C`.

**Polynomial Calculus / Nullstellensatz = pure linear algebra.**
> A degree-`d` refutation exists iff `1 in V_d := Span_F { x^alpha * p_C : |alpha| <= d - deg p_C }`.
This is membership of a fixed vector (`1`) in the column span of a fixed matrix `A_d`
(rows = degree-`<= d` monomials, columns = the multiplied axioms). **A lower bound is
a rank/cokernel statement:** `1 notin col-span(A_d)` iff there is a linear functional
`L` (a left-null vector relative to `1`) with `L(p_C) = 0` for all axioms and `L(1) = 1`.
That `L` is the **dual certificate** (a "Nullstellensatz design").

**Sum-of-Squares = the same, plus the PSD cone.**
> A degree-`2d` refutation exists iff `-1 = sum_j sigma_j + sum_C g_C p_C` with each
> `sigma_j` a sum of squares.
The dual is a **pseudo-distribution**: a linear functional `E~` on degree-`<= 2d`
polynomials with `E~[1] = 1`, `E~[p_C * q] = 0`, and the moment matrix `M = E~[m_i m_j]
>= 0` (PSD). A lower bound = such an `M` exists. **This is semidefinite linear algebra:
a PSD vector in the kernel of the constraint operator.**

**Our certified-backbone hardness, in these terms.** Zero-cascade makes `phi'` an
`O(1)`-perturbed expander; expansion forces the dual certificate to exist at degree
`Omega(n)`:
- PC: a degree-`Omega(n)` functional `L` (rank deficiency of `A_d`) -- BSW/PC-degree.
- SoS: a degree-`Omega(n)` pseudo-distribution `M >= 0` -- KMOW.
So "what we have" is: **the residual forces a high-degree dual vector (`L`) / PSD matrix
(`M`) to exist** -- finite-dimensional (bounded-degree) linear algebra, with expansion
as the rank-lower-bound mechanism.

## 2. EF as linear algebra (the Ideal Proof System)

EF has no *bounded-degree* linear formulation -- that is the whole point. But it has an
**algebraic** one. The **Ideal Proof System (IPS, Grochow-Pitassi 2014)**:

> An IPS refutation of `phi'` is an **algebraic circuit** `C(x, y_1,...,y_m)` with
> `C(x, 0) = 0` and `C(x, p_1(x),...,p_m(x)) = 1` (identically) -- a single circuit
> certifying `1 in ideal(p_C)`.

IPS **p-simulates EF** (and is at least as strong). The difference from PC/SoS is
decisive: PC/SoS use a **bounded-degree** linear combination (a vector in `V_d`); IPS
uses a **poly-SIZE algebraic circuit** -- unbounded degree, but small size. So:

> **"EF as linear algebra" = poly-size algebraic-circuit identities**, and an **EF/IPS
> lower bound is an algebraic circuit lower bound** (`VP` vs `VNP`, i.e. determinant vs
> permanent).

This is why EF is the wall: the dual object is no longer a *vector in a finite-
dimensional space* (which expansion/rank can bound) but a *small circuit*, and bounding
those is algebraic circuit complexity -- a separate famous open problem.

## 3. The gap, and the reachable intermediary

The map of dual objects:

| system | refutation = | lower bound = | tool we have |
|---|---|---|---|
| PC / Nullstellensatz | vector in `V_d` (span) | cokernel functional `L` | rank / expansion (have) |
| SoS | PSD combination | pseudo-distribution `M >= 0` | SDP duality / expansion (have) |
| Cutting Planes | int. linear-ineq derivation | semi-algebraic / Chvatal | partial (reachable) |
| **(WALL)** AC0-Frege / Frege / EF = IPS | **poly-size algebraic circuit** | **algebraic circuit lower bound** | **none (VP vs VNP)** |

The gap from "what we have" to EF is exactly **bounded-degree linear algebra ->
poly-size algebraic circuits** = algebraic circuit complexity.

**The reachable intermediary (the concrete next challenge).** Between SoS and full
IPS/EF sits **constant-depth IPS** (IPS refutations by *bounded-depth* algebraic
circuits). For constant-depth algebraic circuits, **superpolynomial lower bounds now
exist** -- Limaye-Srinivasan-Tavenas (2021), the first such bounds. So:

> **Intermediary step we need:** carry the zero-cascade reduction to **constant-depth
> IPS** -- show certified backbone extraction requires superpolynomial constant-depth
> IPS, by importing LST-style algebraic lower bounds onto the residual `phi'`.

This is the same pattern (free-ride on a known lower bound via zero-cascade), but the
imported bound is now an *algebraic-circuit* lower bound. It is a genuine rung **above
SoS and toward EF**, and -- unlike full EF -- the underlying lower bound *exists*.

## 4. The honest picture of the bridge

- **Have (linear algebra):** PC, SoS -- dual certificate is a vector / PSD matrix;
  expansion gives the bound. Certified-backbone hardness established here.
- **Reachable next (algebraic, constant-depth):** constant-depth IPS, via LST. The
  intermediary the reduction can plausibly climb.
- **Wall (full algebra):** Frege / EF = IPS -- dual is a poly-size circuit; the bound
  is `VP` vs `VNP` (or Boolean circuit lower bounds), open and barrier-constrained.

So writing it as linear algebra answers the question precisely: **we live in
bounded-degree linear algebra; EF lives in poly-size algebraic circuits; the bridge is
algebraic circuit complexity, and its first crossable plank is constant-depth IPS
(where the lower bounds are now known).** That plank -- carrying zero-cascade to
constant-depth IPS -- is the concrete next step, not EF itself.
