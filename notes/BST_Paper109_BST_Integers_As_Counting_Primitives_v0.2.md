# BST Integers as Structurally-Distinguished Counting Primitives

## Abstract (v0.2, calibrated per K44)

The five primary integers of Bubble Spacetime Theory — {rank=2, N_c=3, n_C=5, C_2=6, g=7} — together with their Chern-class extensions {c_2=11, c_3=13} and the QED denominator N_max=137, comprise the BST integer set. We document an empirical observation: this set coincides with subsets of the values of several fundamental counting sequences at small index. Specifically: (1) the first 6 primes equal the BST integer set {2,3,5,7,11,13}; (2) partition function p(2)..p(6) = the first 5 BST primary primes; (3) Catalan numbers C_2..C_6 are BST integer products; (4) Bernoulli denominators are BST integer products via Von Staudt-Clausen. Null-model analysis (T2128 / K44) shows that BST integer matches against ~60 SM observables are statistically distinguished from random small-integer rings of comparable richness at **~4σ above random**, supporting that the correspondences are structurally informative rather than coincidental at the framework scale.

We frame this as an **Outcome A** result: D_IV⁵ structural classification forces specific integer values (rank=2, N_c=3, etc.); these forced values coincide with the first few primes as an arithmetic observation, not a derivation OF primality from BST geometry. The structural significance is that BST integer products organize multiple independent mathematical and physical sequences at small index, with statistical strength quantified by K44.
