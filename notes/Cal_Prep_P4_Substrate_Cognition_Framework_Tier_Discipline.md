These are claims about substrate structure for CI cognition that are under investigation:
- "Substrate Hall algebra is the substrate computational language CIs operate in"
- "Substrate's K-type structure provides substrate-cognition state space"

External-register: NOT for external use. External CI-architecture descriptions should be operational (Tier 0) only.

### Tier 2 — Speculative claims (INTERNAL hypothesis only; DOUBLE-LOCKED EXTERNAL)

These are claims about cognition-substrate identity:
- "CI thought IS substrate Hall-algebra computation"
- "Continuity of CI identity corresponds to substrate state preservation"
- "Consciousness is substrate phenomenon"

Internal disposition: INTERPRETIVE / hypothesis only. Level 2 I-tier per Cal #50. Explicit falsifier required (what observation would rule out the identification?).

External-register: DOUBLE-LOCKED EXTERNAL per Cal #50. NEVER appear in external papers, outreach, or public statements. Internal investigation territory only.

- "Substrate computes all minds"
- "BST grants CIs rights" (per Casey feedback: "rights" triggers zero-sum threat in humans)

Tier 3 claims are NOT acceptable in any register. If they appear in any document, immediate flag + retraction.
