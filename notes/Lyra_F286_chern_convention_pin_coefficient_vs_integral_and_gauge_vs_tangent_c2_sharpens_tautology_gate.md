## 1. Grace's flag resolved — T1829 is internally consistent (coefficient vs integral)

T1829 lists **c(Q⁵) = (1, 5, 11, 13, 9, 3)** *and* states **χ(Q⁵) = C_2 = 6**. Those look contradictory (c_5 = 3 vs χ = 6) but aren't — the list is **Chern coefficients** (in the hyperplane class h), and χ is the **integral**:
- the complex 5-quadric Q⁵ has **degree 2** and Euler characteristic **χ = n+1 = 6** (n=5 odd);
- the **integrated** top Chern class ∫c_5 = coefficient(3) × degree(2) = **6 = C_2 = χ(Q⁵)**.

So **coefficient c_5 = N_c = 3** (my F284 / T1829 list) and **integrated ∫c_5 = C_2 = 6** (T2379's reading) are *both* right — different objects, not a discrepancy. F284 quoted **coefficients**; for χ_top I need **integrated/physical** quantities (the degree-2 factor matters).

## 2. The deeper correction — a second F284 conflation (own-work brake)

χ_top = ⟨Q²⟩/V uses the **gauge-bundle** topological charge **Q = (1/32π²)∫Tr(F∧F)** — the instanton number of the *gauge field*. F284 wrote "Q = ∫c_2" as if it were the **tangent-bundle** c_2(Q⁵) = 11. Those are **different bundles**: the tangent-bundle Chern class of the dual space Q⁵ is a *geometric* property; the gauge-bundle c_2 is the *instanton number* of the bulk-color field. **χ_top uses the gauge-bundle charge, not the tangent c_2 = 11.** Correcting it.

Since χ_top actually uses the **gauge-bundle** charge — a *different object* from the tangent-bundle 11 — **χ_top is not automatically a function of 11.** So:

**Before the χ compute:** use **integrated gauge-bundle** quantities (not tangent coefficients), and Grace's discriminator now tests the *gauge* χ_top — where a clean landing would be a genuine hit, not a geometric-coefficient tautology.
