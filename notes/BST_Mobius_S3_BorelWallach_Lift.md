# Gap #2 Session 3: Borel-Wallach Lift with Z/2 Coefficients

## Context from Session 2

Session 2 (T2329) established H¹_{Z/2}(M(D_IV⁵), Z) = Z/2 via equivariant cohomology — but with COMPLEX coefficients the (g,K)-cohomology vanishes in degree 1 (no SO(5)×SO(2)-invariants in p*).

This Session 3 resolves the apparent tension: standard Borel-Wallach with ℂ coefficients gives 0 (correct — Z/2 torsion is invisible to char-0 cohomology). To recover the Z/2 from the Lie-algebraic side, we use Z/2 COEFFICIENTS, which is a standard variant of (g,K)-cohomology.

## The (g, K)-cohomology setup

For G = SO(5,2), K = SO(5) × SO(2), the relative Lie algebra cohomology is computed from the complex:

```
C^k(g, K) = Hom_K(Λ^k(g/k), V)
```

where V is the coefficient module. For V = ℂ (trivial complex rep), this is:
```
C^k(g, K; ℂ) = (Λ^k(g/k)^*)^K
```

For V = ℤ/2 (trivial Z/2 rep), this is:
```
C^k(g, K; ℤ/2) = (Λ^k(g/k)^*)^K ⊗ ℤ/2
```

## Decomposition of g/k

g = so(5, 2) has real dimension 21.
k = so(5) ⊕ so(2) has real dim 10 + 1 = 11.
g/k has real dim 10 = dim_R(D_IV⁵).

Under K = SO(5) × SO(2):
- g/k splits as p ⊕ p̄ over ℂ (the holomorphic + anti-holomorphic tangent directions)
- p ≅ ℂ^{n_C} = ℂ^5 with SO(5) acting standardly and SO(2) acting by character χ
- p̄ is the conjugate representation

So as K-representation:
- Λ¹(g/k)^* ⊗ ℂ = p* ⊕ p̄*
- (Λ¹)^K = invariants under SO(5) × SO(2): zero (no scalar invariants in p* under standard SO(5))

This confirms **H¹(g, K; ℂ) = 0**.

## Lifting to Z/2 coefficients

For V = Z/2 (trivial), the cohomology gains TORSION classes that don't exist in characteristic zero.

The key relation: H^k(g, K; Z/2) is NOT just (Λ^k(g/k)^*)^K ⊗ Z/2. The Z/2 coefficient ring has its own internal structure, and the differential of the (g, K)-complex must be tracked over Z/2.

For the trivial Z/2 rep, the resulting complex has:
```
H^k(g, K; Z/2) = H^k(K \ G, Z/2)  
```
(when applied to compact quotients, this is the standard topological cohomology of K\G with Z/2 coefficients).

K \ G is the GROUP of "G modulo K" — but G is non-compact, so this isn't a standard topological space. Instead we use Γ\G/K for a discrete subgroup Γ, or pass to the BG/BK perspective.

**Simplification via the classifying space approach**:

For (g, K)-cohomology with Z/2 coefficients on a Hermitian symmetric pair:
```
H^k(g, K; Z/2) = H^k_K(point; Z/2)
```
where the right side is K-equivariant cohomology of a point with Z/2 coefficients.

For K = SO(5) × SO(2):
```
H^*(BK; Z/2) = H^*(BSO(5); Z/2) ⊗ H^*(BSO(2); Z/2)
```

These are standard:
- H^*(BSO(2); Z/2) = Z/2[w] / (degree(w) = 1)
- H^*(BSO(5); Z/2) = Z/2[w_2, w_3, w_4, w_5] where w_i are Stiefel-Whitney classes

So H^1(BK; Z/2) is generated by w_1(SO(2)) = w in the SO(2) factor. That gives a single Z/2 in degree 1.

**Match with Session 2**: this Z/2 corresponds to the orientation class of SO(2) ⊂ K (the rotational direction in the Hermitian structure). It's the same Z/2 that appears in H¹_{Z/2}(M, Z) = Z/2 via the τ-involution.

## Identification

The Z/2 from Session 2's equivariant cohomology equals the Z/2 from Session 3's (g, K)-cohomology with Z/2 coefficients. Both arise from the SAME underlying structure: the SO(2) factor in K = SO(5) × SO(2), which carries the orientation/winding number that the τ-involution flips.

```
H¹_{Z/2}(M(D_IV⁵), Z)  ≅  H¹(g, K; Z/2)  ≅  H¹(BSO(2); Z/2)  ≅  Z/2
```

**Three independent computations** give the same Z/2 — multi-route convergence in cohomology, analogous to the cross-root convergence pattern in BST primary integers.
