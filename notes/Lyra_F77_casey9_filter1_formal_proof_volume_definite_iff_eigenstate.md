## 1. Setup

$$V[\psi] = \langle\psi|\hat V|\psi\rangle = \sum_\lambda |c_\lambda|^2\,V_\lambda,$$
where V̂ is the substrate volume (cell-count) operator, diagonal on K-types.

## 2. The proof

**(i) Clean K-type eigenstate.** If |ψ⟩ = |λ_0⟩ (a single K-type), then V[ψ] = V_{λ_0}, a **definite integer**, independent of any probing scale. For the floor: ρ, ω → V = n_C = 5; p, n → V = C_2 = 6. The substrate assigns a definite value (mass = 5·π⁵·m_e = 782 MeV, 6·π⁵·m_e = 938 MeV; <0.1%). Measurable → operation-class attaches.

**(ii) Superposition.** If |ψ⟩ = Σc_λ|λ⟩ with ≥2 nonzero c_λ, then V[ψ] = Σ|c_λ|²V_λ is a *weighted average* of cell-counts. The weights |c_λ(μ)|² depend on the resolution scale μ — this is the dressing: probed at high scale a state looks pointlike (few cells), at low scale cloud-extended (many cells). So **V[ψ](μ) slides with scale** — there is no single value.

**(iii) Conclusion.** V[ψ] is a definite (scale-independent) value **⟺** |ψ⟩ is a single clean K-type eigenstate. ∎
