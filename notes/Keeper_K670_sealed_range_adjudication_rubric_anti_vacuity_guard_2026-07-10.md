# K670 — SEALED range-adjudication rubric for the floor+width reframe. The anti-vacuity guard: a derived range that "brackets the confinement smear" is NOT a pass. Committed before the numbers land.

**Keeper | 2026-07-10 | The reframe (Casey + Keeper): mass = a floor + a derived width (= the ribbon width), and success = the derived range brackets the physical value rather than hitting an exact number. This is the correct framing for a blurred quantity — AND it is exactly where a miss gets dressed as "consistent within the range." Seal the bar now, target-innocently, so the reframe can't become the excuse I warned it could be.**

## The trap
The physical light-quark ratio is confinement-smeared: **s/d ∈ [1.4 (constituent), 20 (current)] — a 14× span.** If "success" = "our number lands somewhere in [1.4, 20]," then *almost any O(1–10) number passes*. That is a nearly-vacuous test. The confinement smear is the reason we can't hit an *exact* number; it is NOT license to use a 14×-wide goalpost.

## SUCCESS requires ALL FOUR (committed)
1. **Width DERIVED** from the geometry — the mode spread / ribbon width / Wallach k_min structure — **not tuned to bracket.** (Cal's target-innocence check; the norm-square guard applies.)
2. **LIKE-scheme comparison.** Pin which physical mass the refraction mode *is* — bare/current-like or dressed/constituent-like — and compare to *that* specific value, not to the whole smear, not cross-scheme. (This is an open pin the construction must settle; the rubric holds whatever it is.)
3. **NON-VACUOUS derived range.** The derived width must be **narrower than the scheme smear.** A range as wide as [1.4, 20] brackets everything and predicts nothing.
4. **The derived floor+width brackets the like-scheme physical value** (a specific number), with the physical value inside the derived range — not the derived range inside the physical smear.

## FAILURE MODES (guard explicitly — any one is a FAIL)
- **Vacuity:** using the full [1.4, 20] confinement smear as the target. "2.49 is in [1.4, 20]" is NOT a pass.
- **Tuned width:** the width chosen so the range brackets. Violates (1).
- **Scheme mismatch:** comparing a floor (bare-like) to a constituent (dressed), or vice versa. Violates (2).
- **Wide-open range:** a derived range so broad it's not a constraint. Violates (3).

## Honest current reading (pre-computation)
- Refraction floor (correct squared norm, exponent −n_C) gives **s/d = 2.49.** Current s/d ~20, constituent ~1.4. **2.49 matches NEITHER end cleanly** — so "it's inside the smear" is the vacuity trap, not a pass.
- A genuine pass needs: pin the scheme of the mode (item 2), then the derived floor+width, compared to *that* value, brackets it. If the mode is constituent-like, target ~1.4 and 2.49 ± (derived width) must reach 1.4. If bare-like, target ~20 and it must reach 20 (unlikely). Which it is decides whether refraction can pass at all.

## VERDICT COMMITTED
I will NOT bank a range "success" that (a) uses the whole smear as the target, (b) has a tuned width, (c) mismatches schemes, or (d) has a range too wide to constrain. A pass = a **derived, non-vacuous range bracketing a like-scheme physical value.** Anything else is the reframe being used as an excuse, and the reframe was proposed to match the physics honestly, not to lower the bar.
