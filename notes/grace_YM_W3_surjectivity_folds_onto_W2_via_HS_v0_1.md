# YM W3 — surjectivity folds onto the W2 spectrum match via HS

**W3 (= L4) stated precisely:** is the Rehren dual of 4D Yang-Mills the **full** Hardy space H²(D_IV⁵), or only a
**sub-net**? If a sub-net, the gap we computed (the lowest K-type) could be an *extra* bulk state, not a YM
glueball — and the whole conditional "Δ = C₂" would be reporting the wrong object's gap. Two directions:

- **(a)** every YM operator → a D_IV⁵ K-type [the map is well-defined into H²] — the easy direction.
- **(b)** every D_IV⁵ K-type → a YM operator [surjective **onto** H²] — **this is W3 / L4.**

## The HS reduction (T2489 does real work)

The Hardy-Szegő isometry is a *unitary* H²(D_IV⁵) ≅ H²(Shilov boundary) — "the interior IS its boundary values."
Apply it to direction (b):

> **(b) ⟺ every boundary mode is a YM operator ⟺ YM saturates the conformal boundary algebra.**

HS converts an **interior-fullness** question (hard, analytic) into a **boundary-completeness** question (standard).

## The boundary answer, and the honest subtlety

The boundary CFT is *defined* as the UV-conformal limit of YM (asymptotic freedom → UV fixed point). So on the
boundary, YM generates the boundary algebra essentially **by construction**, and HS extends that to the full
interior H². **So surjectivity is not a separate hard analytic problem** — it follows from HS + (boundary = UV-YM),
*provided* the bulk spectral theory is the IR-completion of UV-YM **with no extra states.**

**The load-bearing residual is "no extra/orphan states" — and it is exactly the W2 spectrum match:**

- If the **full** D_IV⁵ Casimir tower = the **full** YM glueball spectrum (every K-type a glueball, no orphan
  D_IV⁵ modes, no missing glueballs) → the dual is **full** → **W3 holds**, and the gap *is* the YM gap.
- So W3 is decided by the **same** spectrum match as W2: **W2 = "the modes that match are YM" (discrimination);
  W3 = "no modes fail to match" (completeness).** Two faces of one check, run in two directions.
