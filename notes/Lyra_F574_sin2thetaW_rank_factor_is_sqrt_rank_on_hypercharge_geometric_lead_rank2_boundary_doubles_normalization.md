## Where it landed

The fermion trace is **locked to the forbidden GUT value**: Tr(T₃²)=2, Tr(Y²)=10/3, Tr(Q²)=16/3 → sin²θ_W = 3/8 — invariant under any decomposition of Y (including left-right Y = T₃^R + (B−L)/2). Since BST is no-GUT (Five-Absence), 3/8 is correctly *not* its value; the geometric normalization must differ.

Solving for what makes it 3/13:
$$\sin^2\theta_W = \frac{\mathrm{Tr}(T_3^2)}{\mathrm{Tr}(T_3^2) + c^2\,\mathrm{Tr}(Y^2)} = \frac{3}{13} \;\Longrightarrow\; c^2 = \text{rank} \ \text{(exactly)}.$$
So **Q = T₃ + √rank·Y** — the hypercharge carries a factor √rank = √2 — and Tr(Q²) = Tr(T₃²) + rank·Tr(Y²) = 26/3 → sin²θ_W = 2/(26/3) = **3/13 ✓**. The c² = rank fell out cleanly; it was not chosen to fit.

The charge circle SO(2)/S¹ lives on that boundary, so the hypercharge is measured across *both* rank directions → its normalization picks up a factor **rank** relative to a single-direction (GUT-like) count:

- **GUT** reads the hypercharge single-direction: Q = T₃ + Y → 3/8.
