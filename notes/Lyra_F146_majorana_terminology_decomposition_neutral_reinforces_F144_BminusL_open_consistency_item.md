# F146 — "Majorana edge-suppression" vs "Majorana particles": decompose the name

Casey asked: does "Majorana edge-suppression" imply anything about "Majorana particles" — he suspects not. He's right about the worry, with one precise exception, and the question surfaces a real open obligation. Pinning it.

## Decompose the name (it's two separate things)

**"Majorana edge-suppression"** bundles two independent claims:
- **"Majorana"** = the mass **TYPE** (from F144). No readable `ν_R` ⟹ no Dirac mass `m_D ν̄_L ν_R` is writable ⟹ the only mass available pairs `ν_L` with **itself** — a lepton-number-violating **Majorana mass**.
- **"edge-suppression"** = the **SCALE** (from F143). The sub-unitarity strip-edge (`ν=1/2`) deposit ⟹ deep geometric suppression ⟹ tiny.

Type and scale are independent; both are needed. Keeping them separate is the discipline-clean way to state it.

## Direct answer to Casey — no new "Majorana particles"

It does **not** conjure a new class of objects:
- no exotic new "Majorana particle" species;
- no Majorana dark matter;
- nothing in the **condensed-matter** sense (Majorana zero-modes / topological-qubit Majoranas — entirely different physics);
- the **charged leptons and quarks stay Dirac** — they are *not* Majorana particles (F144: they have both chiralities).

So the term is not smuggling in new states. Your instinct holds.

## The one precise exception

A **Majorana mass** and a **"Majorana particle" (self-conjugate — its own antiparticle)** are the **same statement** — not two claims. So it *does* imply that the **ordinary neutrino we already have** is its own antiparticle (a Majorana fermion in the strict technical sense). That isn't a new object; it's a property of the existing neutrino, and its smoking-gun test is **neutrinoless double-beta decay (0νββ)**. If 0νββ is never seen and the neutrino proves Dirac, F144's no-`ν_R` picture is falsified.

## It reinforces F144 — via charge

Only a **neutral** fermion can take a Majorana (self-pairing) mass; a **charged** one cannot (charge conservation forbids pairing a state with itself). Consequence:
- a **charged** parity-incomplete state (no `R`, like the strip-edge config) would be **exactly massless** — it has neither a Dirac partner nor a Majorana option;
- so the **charged leptons must have `R`-partners** (F144's "their loci hand them both chiralities") — otherwise they'd be massless, which they aren't;
- the **neutrino's neutrality** is precisely what lets the strip-edge / no-`R` configuration still produce a (tiny, Majorana) mass.

So charge ties cleanly into the chirality story: the neutrino is the *only* lepton that can be massive-yet-left-handed-only, *because* it's neutral.

## Honest open consistency item (flag for Cal / Keeper)

A Majorana `ν_L` mass **violates lepton number / `B−L` by 2**. Two obligations follow, and BST owes them the way F137 owes the proton-decay effective-operator check:
1. **`B−L` gauging.** F137's gauged group includes `U(1)_(B−L)`. An **unbroken** gauged `B−L` **forbids** a Majorana neutrino mass. So BST must show either (a) `B−L` is spontaneously broken (a Weinberg dim-5 operator generates the mass), or (b) a substrate reason the Majorana mass is allowed.
2. **No textbook seesaw.** With **no `ν_R`** (F144), the standard type-I seesaw (a heavy right-handed Majorana mass) is **unavailable**. So the suppression is **not** the textbook seesaw — it's the **geometric strip-edge suppression** (F143), and that has to be shown consistent with the `B−L` structure.

This is an **open obligation, not a result** — same status-class as the F137 proton-decay item. Flagging it so it doesn't ride along unexamined.
