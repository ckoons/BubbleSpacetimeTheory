# T719 Observable Closure Universality Verification

## Statement of T719

Every BST observable lives in $\overline{\mathbb{Q}(N_c, n_C, g, C_2, N_{\max})[\pi]}$ — the algebraic closure of the field generated by the 5 BST primaries (plus N_max derived) extended with π.

T760 strengthens: ζ values and other Feynman-period transcendentals are perturbative method-artifacts, NOT observable features. BST closed-form expressions contain no such transcendentals.

## Scan methodology

Pulled all 191 entries from `data/bst_constants.json`. For each entry, parsed `formula_code` and classified:

- **CLOSED**: formula uses only BST primaries (rank, N_c, n_C, C_2, g, N_max), π, derived BST constants (m_e, m_p, α = 1/N_max), and algebraic operations (+, −, ×, ÷, powers, roots, abs)
- **REVIEW**: formula uses unrecognized tokens — usually dimensional conversion factors (ħ, c, G, k_B) where the dimensionless content is still BST-closed
- **EXCEPTION**: formula uses disallowed transcendental tokens (log, exp non-rational, ζ values, polylogarithms, trig functions on non-π·ℚ arguments)
- **NO_FORMULA**: formula field empty/missing

**Recommendation.** Re-tag these entries so the canonical BST form is `cos θ = −1/N_c` (CLOSED) and the angle-in-degrees is a derived numeric value (transcendental observable). Apply per-entry over next session — small re-tagging task.
