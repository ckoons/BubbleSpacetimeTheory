## 2. Why scale-independence MIGHT be a boundary — the candidate mechanism

Here is the principled reason it *could* be the substrate's edge, and it is program-consistent:

The substrate carries the **4D conformal group SO(4,2) ⊂ SO(5,2)** (the program's proto-AdS/CFT structure). A **conformal** structure is intrinsically **scale-free** — it sits *at a fixed point*, where the beta function vanishes (β = 0 ⟺ conformal). Scale-*dependence* — the running of couplings — is precisely the **breaking** of conformal symmetry, the *departure* from the fixed point (the trace anomaly is the order parameter of that breaking). So:

> If the substrate is **only the fixed-point data** (the conformal, scale-free structure — the boundary CFT in the AdS/CFT reading), then it encodes scale-free structure but **not the flow away from the fixed point** — and *scale-independence is its boundary*, with the running living in a separate (bulk/emergent) layer.

This is the clean candidate for "the substrate is structure, not dynamics": the substrate = the conformal fixed point; running = the departure the substrate doesn't itself contain. It is exactly the AdS/CFT split the program already has (SO(5,2) ⊃ SO(4,2): boundary CFT scale-free, bulk = the flow).

The conformal fixed point provides the UV structure; the content it carries determines *how it flows away* (the anomaly); the flow is a *departure* from the substrate's conformal symmetry whose *rate* the substrate nonetheless *fixes*. "Emergent" is then the wrong word: the running is **substrate-determined**, not a separate layer.

If that holds, **scale-independence is NOT the boundary.** The only genuinely free thing is the **absolute scale** — *where on the flow we sit* — which is the single dimensionful anchor (ℓ_B) every theory takes. The boundary is the one unit, not a kinematics/dynamics seam.
