# F88 — Why exactly three generations: the count is rank + 1

## 0. Grace's bigger prize

## 1. The backbone: a rank-r domain has r+1 boundary-orbit strata (theorem)

**Korányi-Wolf boundary-orbit theorem.** Let D be a bounded symmetric domain of rank r, G its identity component of automorphisms. The closure D̄ decomposes into exactly **r+1 disjoint G-orbits**, indexed by face-rank k = 0, 1, …, r:
- k = r: the open domain D itself (the top stratum);
- k = r−1, …, 1: the intermediate boundary orbits (each boundary face is itself a rank-k bounded symmetric domain);
- k = 0: the rank-0 faces = points = the **Shilov boundary**.

This is standard structure theory (Wolf 1972; Korányi; Faraut-Korányi Ch. VI) — the boundary of a symmetric domain stratifies by face-rank, and there are exactly r+1 strata. It is a **theorem**, not a BST posit.

## 2. D_IV⁵ has rank 2, so exactly 3 = rank+1 strata

D_IV⁵ has **rank 2**. By the theorem, its closure has exactly **r+1 = 2+1 = 3** G-orbit strata:
- **k = 2** (open domain): the **bulk** — full support dim n_C = 5;
- **k = 1** (rank-1 boundary faces): the **Cartan/polydisk** stratum — support dim rank = 2;
- **k = 0** (Shilov boundary): the **points** — support dim 0.

These are exactly F86's three support tiers {n_C, rank, 0} = {5, 2, 0}, which Elie (4074) verified locks to the trichotomy π-ladder (π⁵ / π² / π⁰) independently. So the support-flag of F86 is the Korányi-Wolf orbit stratification, and there are **exactly three tiers because rank = 2**.

## 3. Each generation = one stratum ⟹ exactly rank+1 = 3 generations

F86 identified each lepton generation with one support tier (electron = bulk, muon = Cartan, tau = Shilov point), 4-lane coherent (support-flag = trichotomy = π-ladder = lepton-mass steps). Combined with the theorem:

The "3" of the Standard Model is **rank + 1**, where rank = 2 is the rank of the substrate domain. This converts the generation count from an unexplained input integer into a substrate consequence.

The SM has **3 generations of quarks AND 3 of leptons** — the same count, with no SM reason they must match. F88 explains it: the generation count is a property of the **substrate domain** (its rank-r boundary stratification), not of the fermion species. Quarks and leptons live on the **same** D_IV⁵, so both inherit the **same** rank+1 = 3 strata.
